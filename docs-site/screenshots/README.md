# Screenshots Directory

Place your DocForge screenshots here for the user guide documentation.

## Required Screenshots

1. **step1-url-input.png** - The main URL input field on the landing page
2. **step2-analysis.png** - Website analysis showing detected strategy
3. **step3-options.png** - Configuration options (crawl strategy, page limits)
4. **step4-progress.png** - Crawling progress with logs
5. **step5-preview.png** - PDF preview modal
6. **step6-download.png** - Download confirmation/button

## Screenshot Guidelines

- **Resolution**: 1920x1080 or similar high-quality resolution
- **Format**: PNG for better quality (JPG acceptable)
- **Content**: Show the actual DocForge interface in use
- **Annotations**: Optional arrows or highlights to emphasize key features

## How to Capture

1. Run DocForge locally
2. Use a screenshot tool (Windows Snipping Tool, macOS Screenshot, etc.)
3. Capture each step of the user journey
4. Save with the appropriate filename
5. Commit to this directory

## Fallback

If screenshots are missing, the documentation will show placeholder images with descriptive text.
